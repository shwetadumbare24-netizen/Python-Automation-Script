# Python Automation Script

## 📌 Project Overview

This project is a Python-based File Organizer that automatically sorts files into different folders based on their file extensions.

## 🎯 Objective

To automate file management and reduce manual effort by organizing files into categorized folders.

## 🛠️ Technologies Used

* Python
* os Module
* shutil Module

## ✨ Features

* Automatically scans a selected folder.
* Creates folders based on file types.
* Moves files into their respective folders.
* Generates a log file containing file movement details.
* Handles errors using exception handling.

## 📂 Project Files

* organizer.py → Main Python script
* logs.txt → Log file generated after execution
* Code.PNG → Script screenshot
* Output.PNG → Program execution screenshot
* LogsFile.PNG → Log file screenshot

## ▶️ How to Run

1. Install Python.
2. Open the project folder in VS Code.
3. Run:
   python organizer.py
4. Enter the folder path when prompted.
5. Files will be organized automatically.

## 📸 Output

The script successfully organizes files into separate folders and records all actions in logs.txt.

## ✅ Conclusion

This project demonstrates Python automation using file handling, folder management, and exception handling. It improves productivity by automating repetitive file organization tasks.
